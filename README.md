\# YT\_MP3


A LOCAL YouTube → MP3 converter built using `yt-dlp`.


Because relying on random online converters is unreliable, unsafe, and cringe.

---

## Features

- Convert YouTube videos to MP3

- Simple Windows launcher (`RUN_CONVERTER.bat`)

- Uses a Python virtual environment for isolation

- Outputs files to a dedicated `MP3` folder

- No ads, no tracking, no uploads

---

## Requirements

- \*\*Windows\*\*

- \*\*Python 3.9+\*\*

- Internet connection

---

## Project Structure

- YT_MP3

-- .venv # Python virtual environment (not committed)

-- MP3 # Output folder for converted MP3s

-- RUN_CONVERTER.bat # One-click launcher
